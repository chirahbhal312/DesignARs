# AR for Rural Education

## Overview
A research-based project exploring how **Augmented Reality (AR)** and **UX design** can improve learning in rural schools by making education interactive and accessible.

## Problem Statement
Rural schools lack interactive tools and infrastructure. Existing AR apps are complex, require high-end devices, and are not suitable for rural users.

## Objectives
- Study digital literacy in rural areas  
- Evaluate AR in education (Science/Geography)  
- Design user-friendly UX (Figma)  
- Propose offline-first solutions  
- Validate using usability testing  

## Tech Stack
- Figma, Miro  
- Design Thinking  
- User Research (Interviews, Personas)  
- Usability Testing  

## Methodology
Empathize → Define → Ideate → Prototype → Test
